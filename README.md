# Rotten Potatoes

## Production Server
https://rotten-potatoes-vn.herokuapp.com/

## Technologies
- Node.js
- Express/Handlebars
- MongoDB

## Usage
To run this project in your local directory, follow the commands below.
```bash
git clone https://github.com/vladyslavnUA/potatoes.git && cd potatoes/
npm i
nodemon

Head to: http://localhost:8002/
```
